# Trust & Transparency

AuditMate is built on the principle that **audit tools must themselves be auditable**.

---

# Offline by Design

AuditMate requires:

* No internet access
* No cloud services
* No background agents

All collection happens locally at runtime.

---

# Explicit Behavior

AuditMate:

* Executes OS commands
* Reads system state
* Writes local output
* Exits immediately

If AuditMate is not running, it does nothing.

---

# Deterministic Output

Given:

* Same system state
* Successful data collection

AuditMate produces consistent output.

---

# Non-Opinionated Design

AuditMate does **not**:

* Judge system configuration
* Score security posture
* Filter "noise"
* Interpret results

AuditMate records **facts only**.

This allows:

* Independent review
* Audit transparency
* Forensic reliability

---

# Open Source Core

Free tier:

* Fully open source
* Auditable collectors
* Transparent behavior

---

# Free vs Professional

Both tiers:

* Use identical collectors
* Observe same system state

Professional tier adds:

* Evidence signing
* Bundle export
* License management

No hidden collectors.

---

# Intended Audience

AuditMate is designed for:

* System administrators
* Security teams
* Compliance teams
* Incident response
* Forensics

AuditMate is not designed for offensive security.
