# Trust Model

AuditMate is designed to be a deterministic and verifiable system state recorder.

---

## Offline First

- no network communication
- no telemetry
- no external dependencies at runtime

---

## Deterministic Output

Given the same system state, AuditMate produces consistent results.

---

## Transparency

- free version is fully open source
- collectors are inspectable
- output is locally generated

---

## Principle

AuditMate does not interpret systems.

It records them.
